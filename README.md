## ğŸ¨ Customization

### Change Colors

Edit `css/style.css` and modify the CSS variables at the top:

```css
:root {
    --bg-primary: #050508;      /* Main background */
    --bg-secondary: #0a0a0f;    /* Secondary background */
    --accent: #C4B5FD;          /* Accent color (lavender) */
    --text-primary: #FAFAFA;    /* Main text */
    --text-secondary: #A1A1AA;  /* Secondary text */
}
```

### Update Content

Edit `index.html` to change:
- Name and title in the Hero section
- Business/Education/Systems descriptions
- Contact information (WhatsApp numbers, email)
- Focus areas content

### Replace Images

1. Add your new images to the `images/` folder
2. Update the `src` attributes in `index.html`
